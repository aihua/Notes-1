The one minute manager


提醒我们使用一分钟的时间，正面审视那些被我们管理的人，
    认识到他们才是我们最重要的资源

星期会议:
    定期对上一个星期的工作进行总结分析，包括成果，问题以及需要完成的事情。
    为下个星期制定计划和工作策略

偏重于工作结果，还是偏重于人

结果是通过人来获得的

自我感觉良好的人，才能做出优秀的业绩

帮助别人获得良好的自我感觉，就是提高效率的关键


效率
    数量和质量

一分钟目标
    !!! 需要团队对这个目标达成一致

    !!! 让员工了解好的表现是怎样的
        明确衡量工作表现的标准
        Java code standard

    !!! 把每个目标写在一张纸上, 一分钟内可以把这个目标读一遍
        目标简洁，清晰，明确
        80%的工作成果来源于20%的目标

    !!! 每天使用一分钟时间审视自己的表现
        看看自己的行为是否与目标一致

    根据目标定期检查工作进度

    从行为角度解释问题
        从可观察，可测量的角度描述问题
    从行为角度描述期望的事情是怎样的
    现实与期望之间的差距

    1. 将解决方案先一一列举出来
        画到白板上
        写道纸上

        不论该方案看上去多么简单或多么不靠谱
        
    2. 从列举的方案中选择，组合方案

    一分钟目标举例:
        找出工作中存在的问题，并找到有效的解决办法

一分钟称赞

一分钟批评
