Discuz！论坛提供有安装步骤（http://faq.comsenz.com/library/system/env/env_linux.htm），
但是教程太老，步骤太多，太繁琐且费时，按照步骤操作很可能还会出问题，
所以不建议按照官方论坛推出的教程配置Discuz运行环境，现提供最简单的安装步骤：

1、安装LAMP（其他linux系统，参考http://en.wikipedia.org/wiki/LAMP_%28software_bundle%29）：
只需两条语句即可：
sudo apt-get install tasksel；
sudo tasksel install lamp-server(参考文档https://help.ubuntu.com/community/ApacheMySQLPHP).

2、下载Discuz！，http://www.comsenz.com/downloads/install/discuzx（Ubuntu下载UTF-8版本）
3、安装Discuz！，参考官方步骤，http://www.discuz.net/thread-2141484-1-1.html。


重点难点是第三步，尤其是新手，几乎不理解什么意思，最主要是重点理解括号注释（其中 http://www.domain.com/bbs 为你的站点访问地址），理解以后就很容易安装了，意思也就是说要把地址改成你的apache地址，bbs是建议的文件夹命名，bbs里的内容是解压出的upload内容（可以在apache目录下建立文件夹bbs，然后把upload里的内容复制进去，最后配置权限即可),所列出的文件的权限都需要是777，文件就在bbs目录下。

如果电脑之前安装有mysql，则必须要完全卸载才能继续安装，否则会造成冲突，无法正常安装，
很多时候即便安装成功也很难和php链接起来，如果按照其他教程安装了mysql，而有无法卸载干净，建议重装系统
