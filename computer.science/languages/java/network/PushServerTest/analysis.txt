PC上连接实验

    握手消息: 464 bytes
    心跳消息: 8 bytes
    普通消息：8 bytes附加协议字段
    (上面消息大小包括了服务器和客户端发送和接收的消息)

    使用LAN连接服务器，拔掉网线， 断开连接30秒钟。然后连接网线后， 心跳消息恢复正常。
    使用LAN连接服务器，将网络连接切换到wifi, 无法发送心跳。网络切换到LAN后，心跳消息恢复正常。
